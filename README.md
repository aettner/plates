# Plates

Plates is an Android app for tracking German license plates. It allows you to keep a list of all the license plates you've seen, and it displays your progress on a map of Germany.

## Features

*   **List of all German license plates:** The app includes a comprehensive list of all German license plate codes, along with their corresponding city and state.
*   **Tracking seen plates:** You can mark plates as "seen" to keep track of your progress.
*   **Filtering and searching:** You can filter the list of plates to show all, seen, or unseen plates. You can also search for specific plates by their code.
*   **Map visualization:** The app displays a map of Germany, with each state colored based on the percentage of seen plates in that state.
*   **Theme selection:** You can choose between light, dark, and system default themes.

## Building and Running

To build and run the app, you'll need to have Android Studio installed.

1.  Clone the repository:
    ```
    git clone https://github.com/your-username/plates.git
    ```
2.  Open the project in Android Studio.
3.  Build and run the app on an emulator or a physical device.

## Technologies Used

*   [Kotlin](https://kotlinlang.org/)
*   [Jetpack Compose](https://developer.android.com/jetpack/compose) for the UI
*   [Kotlinx Serialization](https://github.com/Kotlin/kotlinx.serialization) for JSON parsing
*   [Coil](https://coil-kt.github.io/coil/) for image loading
